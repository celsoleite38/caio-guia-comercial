Caio Guia
Plataforma de guia comercial local para divulgação de negócios e profissionais liberais. Sistema completo com painel administrativo, moderação de anúncios e filtros por cidade/categoria.

Python
Django
Bootstrap
License

🚀 Funcionalidades
Cadastro de Anúncios: Usuários podem cadastrar negócios com logo, descrição, contato e localização
Sistema de Aprovação: Admin modera todos os anúncios antes da publicação
Filtro por Cidade: Suporte a múltiplas cidades com select dinâmico
Planos Free e Pago: Anúncios destaque aparecem primeiro na busca
Busca Inteligente: Pesquisa por nome, descrição, categoria e cidade
Painel Admin: Dashboard com estatísticas, aprovação rápida e gestão completa
Responsivo: Interface mobile-first com tema dark moderno
Contador de Views: Ranking dos anúncios mais visualizados
🛠️ Tecnologias
Backend: Django 5.0 + SQLite
Frontend: HTML5, CSS3, Bootstrap 5, Font Awesome
Autenticação: Django Auth com sistema de perfis
Upload: Pillow para tratamento de imagens
Deploy: Compatível com Render, Railway, PythonAnywhere
